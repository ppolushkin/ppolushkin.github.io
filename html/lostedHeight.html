Lost height CSS

Here is demonstration how to fix height of div .p. The problem is that browser firstly draw "normal" elements (div .p) and then "float" ones (which is div .c in example).
That's why without tricky hack with .p:after height of .p is zero! 

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>p div height is zero</title>
    <style type="text/css">

        .p {
            opacity: 1;
            background-color: #22b333;
            border: 1px dashed #999966;
        }

        .p:after {
            content: ' ';
            clear: left;
            display: block;
        }

        .c {
            height: 100px;
            width: 100px;
            opacity: 1;
            background-color: #999966;
            border: 1px dashed;
            float: left;
        }

    </style>
</head>

<body>

  <div class='p'>
    <div class='c'>

    </div>
    <div class="c">
    </div>
  </div>

</body>
</html>
