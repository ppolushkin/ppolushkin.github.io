Stacking of normal and positioned elements HTML positioning

Positioned (absolute, relative, Fixed) elements are always on top of normal (static)

More info:

  • https://developer.mozilla.org/en-US/docs/Web/Guide/CSS/Understanding_z_index/Stacking_without_z-index
  • https://www.safaribooksonline.com/library/view/css3-the-missing/9781449339456/ch15.html#stacking_elements 

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Stacking without z-index</title>
    <style type="text/css">

        .a {
            height: 100px;
            width: 100px;
            opacity: 1;
            background-color: #22b333;
            border: 1px dashed #999966;
            position: absolute;
            left: 90px;
            top: 20px;
        }

        .a:before {
            content: 'i am absolute or positioned';
        }

        .n {
            height: 100px;
            width: 100px;
            opacity: 1;
            background-color: #999966;
            border: 1px dashed #999966;
        }

        .n:before {
            content: 'i am static or default';
        }

    </style>
</head>

<body>
<div class='a'>
</div>

<div class="n">
</div>

</body>
</html>
