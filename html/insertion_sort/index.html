Insertion Sort

    static void insertSort(int[] dest) {
        int low = 0;
        int high = dest.length;
        for (int i = low; i < high; i++) {
            for (int j = i; j > low && dest[j - 1] > dest[j]; j--) {
                swap(dest, j, j - 1);
            }
        }
    }

    static void swap(int dest[], int l, int r) {
        int t = dest[l];
        dest[l] = dest[r];
        dest[r] = t;
    }
Data structure Array
Worst-case performance О(n2) comparisons, swaps
Best-case performance O(n) comparisons, O(1) swaps
Average performance О(n2) comparisons, swaps
Worst-case space complexity О(n) total, O(1) auxiliary

Possible improvement

Insert position can be find by binary search.
